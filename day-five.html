1. HTML Images - the img tag

  • HTML images have the src attribute which is used to get the source of the image to display on the portion of the webpage specified.
  • They also have the width and height attribute which also helps to define the size of the image.
  • The width and height could be written in numbers or in pixels which we will come to learn of in CSS.
  • They also have the alt attribute which is used to tell more about the image (for browsers that could not load the image, e.g due to bad network or phone memory like we have in the button phones we all once used to surf the net).
  • You can also insert images from another website or from the internet by simply setting the src value of the img tag to the url of the desired image.

2. HTML Tables - the table tag

  • The HTML tables allow web authors to arrange data like text, images, links, other tables, etc. into rows and columns of cells. The HTML tables are created using the table tag in which the tr tag is used to create table rows and td tag is used to create data cells.
  • There are two attributes called cellpadding and cellspacing which you will use to adjust the white space in your table cells. The cellspacing attribute defines the width of the border, while cellpadding represents the distance between cell borders and the content within a cell.
  • You will use colspan attribute if you want to merge two or more columns into a single column. Similar way you will use rowspan if you want to merge two or more rows.
  • Try playing with the table atributes to see how they affect the visual appearance of the table

4. HTML Hyper Links - the anchor (a) tag

  • A webpage can contain various links that take you directly to other pages and even specific parts of a given page. These links are known as hyperlinks
  • Hyperlinks allow visitors to navigate between Web sites by clicking on words, phrases, and images. Thus you can create hyperlinks using text or images available on a webpage, by simply nesting the text, image or any other tag with the anchor tag.
  • A link is specified using HTML a tag. This tag is called anchor tag and anything between the opening a tag and the closing a tag becomes part of the link and a user can click that part to reach to the linked document. Let's try following example which links http://www.w3schools.com at your page.
    • Click me to visit w3Schools!
  • The anchor tag is not only used to link external websites. They can be used to navigate from one page to another on our own website by simply providing the relative path to the respective file we want to vie, be it another html file or an image file, or any sort of file that can be displayed by the browser. For example, let's visit the about page... About Page | Opens an image
  • We have used target attribute in our previous example. This attribute is used to specify the location where linked document is opened. Following are the possible options:
    • Opens in New | Opens in Self | Opens in Parent | Opens in Body
  • You can create a link to a particular section of a given webpage by using name attribute. This is a two-step process.

    First create a link to the place where you want to reach with-in a webpage and name it using a tag with a name attribute.

    Second step is to create a hyperlink to link the document and place where you want to reach as follows Go to the Top

6. HTML Forms - the form tag

  • HTML Forms are required, when you want to collect some data from the site visitor. For example, during user registration you would like to collect information such as name, email address, credit card, etc.
  • A form will take input from the site visitor and then will post it to a back-end application. The back-end application will perform required processing on the passed data based on defined business logic inside the application.
